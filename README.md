# 📝 Lekhak – The Smart Notepad

Lekhak is a **modern and lightweight smart notepad application** designed to help users quickly capture, edit, and organize their thoughts, ideas, and tasks.
It provides a simple yet powerful writing experience with a clean interface focused on productivity and ease of use.

---

## ✨ Features

* 📝 **Create and Edit Notes** – Quickly write and update notes anytime.
* ⚡ **Fast & Lightweight** – Smooth performance with minimal resource usage.
* 📂 **Organized Notes** – Manage your notes efficiently.
* 🎨 **Clean UI** – Minimal and distraction-free writing environment.
* 🔍 **Easy Access** – Quickly view and edit saved notes.
* 💾 **Local Storage** – Notes are stored locally for fast access.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/D-pex/Lekhak-The-Smart-Notepad.git
```

### 2️⃣ Open in Android Studio

1. Open **Android Studio**
2. Click **Open an Existing Project**
3. Select the cloned repository

### 3️⃣ Run the App

* Connect an Android device or start an emulator
* Click **Run ▶**
